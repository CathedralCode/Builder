## Installing Inane Classes

### Requirements

- PHP \>= 7.0
- zendframework/zend-http >= 2.8

### Installation

```shell
php composer.phar require inanepain/inane
```

#### OR

Stuff to add to `composer.json`

```json
"require": {
    "inanepain/inane" : ">=0",
}
```

then simply run:

```shell
php composer.phar update
```
